Formatações Adicionais

Existem também alguns codigos que colocamos para fazer algumas marcações nos sites no texto.

Texto Marcado
É como se fosse um marca texto, no qual podemos destacar parte de um texto mara isso usamos a tag mark

Texto Grande
Como o proprio nome diz, deixa o texto grande, (nesse caso, a tag big é uma tag que esta em desuso, por isso ela esta em vermelho nos codigos)usando a tag big.

Texto Pequeno
Como o nome sugere, esta tag acima, deixa o texto pequeno, ja que small em ingles é pequeno.

Texto Deletado
Neste caso o texto fica riscadinho, como se tivesse sido deletado do texto o qual esta inserido. Para isso usamos a tag del

Texto Inserido
Neste caso, é como se fosse um sublinhado, o que sugere que o texto seria adicionado. Usamos a tag ins para adicionar o efeito de insercao.

Texto Sublinhado
Nesse caso o proprio nome diz, o texto fica sublinhado usando a tag u

Texto Sobrescrito
Este caso é muito usando para se colocar o exponenxial em formulas, ou seja o texto fica em cima por exemplo, na equação seguinte: x2x+8x + 4 x 2. tudo isso usando a ta sup

Texto Subscrito
Neste caso é muito comum usar em formula quimicas como por exemplo a formula da glicose: C6H12O6. Para isso usamos a tag sub
